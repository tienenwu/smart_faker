# SmartFaker Examples

This directory contains comprehensive examples demonstrating how to use the SmartFaker package for generating test data in Flutter and Dart applications.

## Examples

### Basic Usage
- Simple data generation
- Person, internet, location, and company data
- Quick start examples

### Localization
- Generate data in multiple languages
- Support for English, Traditional Chinese, and Japanese
- Locale-specific formatting

### Seeded Generation
- Reproducible random data
- Consistent test data across runs
- Perfect for testing and development

### Schema-based Generation
- Define custom data schemas
- Generate complex data structures
- Maintain data relationships

### Schema Imports
- Import JSON Schema, OpenAPI, and Prisma models
- Auto-map fields, constraints, and relationships
- `dart run example/schema_import_example.dart`

### CLI Utilities
- Launch mock API scenarios from the terminal
- Generate fixtures directly from schema files
- `dart run bin/smart_faker_cli.dart --help`

### Smart Relationships
- One-to-many relationships
- Many-to-many relationships
- Hierarchical data structures

### Batch Generation
- Generate multiple items efficiently
- Create test datasets
- Consistent related data

## Running the Examples

```bash
dart run example/smart_faker_example.dart
```
